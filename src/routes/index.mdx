# Welcome to Qwik Docs

Qwik Docs is a modern documentation framework built with Qwik, designed to create fast, efficient, and user-friendly documentation websites.

- Built on top of Qwik's resumable architecture, Qwik Docs delivers instant page loads and optimal performance while maintaining a rich interactive experience.

## Key Features

- **Resumability**: Leverages Qwik's unique resumable architecture for instant page loads.
- **Zero Hydration**: No JavaScript hydration cost on page load.
- **MDX Support**: Write documentation using MDX for enhanced content creation.
- **Developer-Friendly**: Simple setup and intuitive content organization.
- **Customizable**: Flexible theming and layout options to match your brand.
